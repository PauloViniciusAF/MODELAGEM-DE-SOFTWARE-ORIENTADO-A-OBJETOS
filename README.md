# Modelagem de Sistemas Orientados a Objetos – CC6522

## Descrição
Uma rede de cinemas deseja informatizar o processo de reserva de ingressos em suas diversas
unidades. O objetivo é criar um sistema que permita que os clientes consultem os filmes em cartaz,
escolham sessões e assentos disponíveis, realizem reservas e recebam seus ingressos de forma
prática e segura.
O sistema deve também permitir que administradores gerenciem filmes, sessões e salas em diferentes
locais, garantindo que os assentos não sejam reservados em duplicidade e que o controle de
disponibilidade seja confiável.
Além disso, a rede de cinemas possui um programa de fidelidade com benefícios especiais:

- Clientes podem reservar os melhores assentos se realizarem a compra com até dois dias de
antecedência.
- Cada cliente cadastrado no programa recebe um ingresso gratuito no mês de seu
aniversário.

O sistema deve ainda contemplar a comercialização de ingressos de meia-entrada, conforme
legislação vigente, para:
- Estudantes devidamente identificados.
- Pessoas maiores de 65 anos.
- Professores da rede pública e privada.

## Escopo

- Clientes poderão consultar filmes e sessões disponíveis em diferentes salas e locais.
- Clientes poderão reservar assentos e obter ingressos digitais ou impressos.
-  Administradores poderão cadastrar e gerenciar filmes, sessões e salas em toda a rede.
-  Sistema deve controlar a disponibilidade de assentos em tempo real.
-  O programa de fidelidade deve aplicar regras especiais de reserva e benefícios.
-  O sistema deve gerar relatórios de vendas e renda por sessão.
-  O sistema deve enviar relatórios obrigatórios para a ANCINE (Agência Nacional do Cinema).
-  O sistema deve registrar, para cada sessão, a quantidade de ingressos vendidos e a
renda obtida.
- O sistema deve permitir a emissão de relatórios consolidados por filme, sala, cinema e
período.
- O sistema deve gerar relatórios automáticos após o término de cada sessão.
- O sistema deve manter histórico de reservas e vendas para auditoria.
- O sistema deve registrar a quantidade de ingressos de meia-entrada vendidos por
categoria (estudantes, maiores de 65 anos, professores).
- O sistema deve aplicar corretamente regras de meia-entrada.
- O sistema deve integrar-se com um serviço da ANCINE para obter a lista oficial de filmes
disponíveis.
- O sistema deve registrar quais títulos foram disponibilizados em cada sala, com base na
lista oficial da ANCINE.
- O administrador poderá selecionar títulos dessa lista e disponibilizá-los em suas salas.
